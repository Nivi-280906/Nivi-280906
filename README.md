### 🔐 SkillVault — Digital Skill Portfolio Platform
*Full-Stack Developer · Dec 2025 – Feb 2026*

A full-stack certificate management platform with AI-powered verification and real-time data sync.

**Stack:** JavaScript (ES6+) · HTML5 · CSS3 · Firebase Auth · Firebase Realtime DB · Cloudinary · Vercel

**Highlights**
- Role-based authentication with Email + Google Sign-In
- Real-time CRUD powered by Firebase Realtime DB
- AI-powered credential verification via Cloudinary
- Access-controlled shareable profile links for recruiter-facing credential sharing
- 5+ organic users post-launch, listed on Product Hunt, indexed on Google within 2 weeks
